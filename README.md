## Installation
1. Make sure you have Node.js installed ([download](https://nodejs.org/en/))
2. Clone this repo into your directory
```
git clone git@github.com:chomieu/Flex-Fit.git
```
3. Install any dependencies
```
npm install *dependency*
```
